# 竞赛证书智能识别与信息管理平台

## 项目简介
基于 Streamlit 开发的轻量化竞赛证书智能识别系统，集成 **用户认证与管理**、**Excel 批量导入用户**、**证书信息 OCR 自动提取** 等核心功能，支持管理员统一管控用户权限，学生/教师端上传证书并自动解析关键信息，适配高校竞赛证书管理场景。

### 核心功能
| 模块         | 功能描述                                                                 |
|--------------|--------------------------------------------------------------------------|
| 用户管理     | 管理员可创建/禁用用户、批量导入用户、按角色筛选用户                       |
| 权限控制     | 区分管理员/学生/教师角色，不同角色可见不同功能模块                       |
| 批量导入     | 支持 Excel 批量导入用户，自动校验学工号格式、重复账号等异常               |
| 证书识别     | 上传图片/PDF 格式证书，通过 OCR 自动提取竞赛名称、获奖等级、姓名等信息     |
| 数据存储     | 基于 SQLite 轻量数据库，存储用户信息、证书识别结果等数据                 |

## 环境部署
### 1. 环境要求
- Python 3.8+（推荐 3.10）
- 操作系统：Windows 10/11、Linux、macOS
- 依赖：Tesseract OCR（证书识别必需）
